\section{Group Members Contribution}

% For each group member, describe in detail the contribution to the project.

\begin{description}

    \item[Laszlo] Developed the \texttt{profile.jsp} page, which handles the full user profile view and editing interface.
    Implemented the \texttt{CourseSummaryServlet} to retrieve and display course-related summaries,
    the \texttt{FavoritesPageServlet} to serve the favorites browsing page,
    and the \texttt{RegisterServlet} to manage new user registration logic.
    Also implemented the \texttt{DownloadNoteServlet}, responsible for handling note file downloads.

    \item[Edoardo] Developed the \texttt{home.jsp} page, the main landing page aggregating note listings and course information.
    Implemented the \texttt{FavoriteDAO}, the data access layer for all favorites-related database operations.
    Also developed the \texttt{RemoveFavoriteServlet} for unfavoriting notes,
    the \texttt{ProfileServlet} to serve profile data to the view,
    and the \texttt{DeleteNoteServlet} for note deletion with authorization checks.

    \item[Luca Dusi] Developed the \texttt{upload.jsp} page providing the note upload interface,
    and the \texttt{register.jsp} page for the user registration form.
    Implemented the \texttt{StorageService} utility, which abstracts file storage operations for uploaded notes.
    Also developed the \texttt{UploadNoteServlet} to handle multipart file upload requests
    and the \texttt{FavoriteServlet} to manage the addition of notes to user favorites.

    \item[Milos] Developed the \texttt{favorites.jsp} page, displaying the list of a user's saved notes. Implemented the \texttt{main.js} client-side script providing dynamic UI interactions across the application. Also developed the \texttt{Favorite} model class representing the favorite entity, the \texttt{RatingServlet} to handle note rating submission and retrieval, and the \texttt{UpdateNoteServlet} to manage note metadata updates.

\end{description}
