\section{Objectives}
The goal of this project is to design and develop a server-side Web application called
\textbf{Lecture Notes App}: a platform that allows university students to upload, share,
search, and download lecture notes in PDF format.
Users can register and log in to the platform, organize notes by course, mark notes
as favourites, and rate content shared by other students.

The main objectives of this project are:
\begin{itemize}
    \item Provide a platform for sharing lecture notes
    \item Enable efficient search and organization of content
    \item Allow user interaction through ratings and favourites
    \item Ensure scalability and maintainability of the system
\end{itemize}

\subsection{Technologies Used}

The application is developed using:
\begin{itemize}
    \item Java Servlets and JSP for server-side logic and dynamic content generation
    \item PostgreSQL as the relational database management system
    \item Apache Tomcat as the deployment server
\end{itemize}

\subsection{Motivation}

University students often struggle to find well-organized and reliable lecture notes.
Existing solutions are either fragmented or lack quality control. This project aims to
provide a centralized and user-driven platform to improve accessibility and collaboration.
